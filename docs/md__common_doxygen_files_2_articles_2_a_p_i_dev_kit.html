General API Development Kit 28

The General API Development Kit enables software developers to extend the functionality of Archicad.

The following documents give a short overview of the possibilities and requirements.

  • API White Paper
  • License Agreement
  • Registration

Add-ons built with this API version are compatible with Archicad 28. You can check what improvements and corrections have been made to the former API versions.

  • New API Features in Archicad 28
  • Compatibiliy

Note, that because of the interface changes significantly release by release, we do not ensure neither interface nor binary compatibility with previous versions. So add-ons compiled with older DevKits cannot be loaded in the Archicad 28.

System Requirements

This Developer Kit contains all the necessary support to develop your add-on either on Windows or macOS. This development kit also fully supports cross platform development.

Recommended hardware and operating systems

You can find details on Help Center .

Build environments

  • Windows: Microsoft Visual C++ 2019 Build Tools (v142) installed. Only the 64-bit architecture is supported.
  • macOS: Xcode 12.5+.
