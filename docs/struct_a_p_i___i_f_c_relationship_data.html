Detailed Description

Contains IFC Entity relations which are specific for the IFC. The IFC Entities are represented by their GlobalId.

Remarks
This structure is used in API_IFCDifferenceGeneratorState, ACAPI_Markup_ImportFromBCF and ACAPI_ProjectOperation_Save. The IFC containment tree (containmentTable) is in "child ->parent" format. Example: IfcBuilding ->IfcProject IfcSite ->IfcBuilding IfcBuildingStorey ->IfcBuilding IfcWall ->IfcBuildingStorey IfcBuildingElementPart ->IfcWall. The IFC assignmentTable set (assignmentTable) is in "child ->parent" format. Example: IfcWall ->IfcRelAssignsToGroup IfcRelAssignsToGroup ->IfcGroup. The IFC Type Products (typeProductTable) is in "child ->parent" format. Example: IfcWall ->IfcWallType.
