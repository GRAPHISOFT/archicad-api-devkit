Detailed Description

The Port is associated to an element and it represents a point between elements.

Since
Archicad 27

◆ IsPhysicallyConnectable()

Result< bool > ACAPI::MEP::PortBase::IsPhysicallyConnectable ( const UniqueID & portId) const

Checks if the Port is physically connectable to another Port. Beside

  • Domain
  • MEP System
  • Position
  • Orientation which are only enough to obtain a logical connection, there are a few other Port settings that need to match perfectly in order to obtain a Port-Port physical connection and they are the following:
  • Shape
  • Dimensions
  • ConnectionType (this is a GDL parameter, so it can be checked and modified through API_AddParType)
    Parameters
    portIdThe UniqueID of the Port to check against.
    Returns
    Returns an error if the parameter does not identify a Port, otherwise the result of the check.
