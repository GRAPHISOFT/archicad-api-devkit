Detailed Description

Parameters of the change story settings command.

Remarks
The command is related to an existing story. It must be referred by the index field of the structure. Possible operations are listed in the API_StoryCmdID list. Note, that in case story operations are disabled in team projects; except changing the active story. Note that story operations —except changing the active story— are disabled in teamwork projects.
