Detailed Description

Representation of the display only options.

Remarks
These settings correspond to the relevant items in the View/On-Screen View Options menu. Call the ACAPI_Navigator_GetViewOptions or the ACAPI_Navigator_ChangeViewOptions functions to operate on this data.
