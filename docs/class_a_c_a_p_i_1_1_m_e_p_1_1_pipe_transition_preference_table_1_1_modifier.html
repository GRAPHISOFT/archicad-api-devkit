◆ SetBeginDiameterByKey()

Result< void > ACAPI::MEP::PipeTransitionPreferenceTable::Modifier::SetBeginDiameterByKey ( const Key & key,
double diameter )

Sets the begin diameter value of a table's row.

Parameters
keyThe key that uniquely identifies a row in the PipeTransitionPreferenceTable.
diameterThe value to be set. Must be a non-zero positive number.

If no row exists for the given key, but the key contains valid referenceIds, it inserts the row.

Returns
Returns nothing if the operation was successful, or returns error:
  • if at least one referenceId from the key is not in the PipeReferenceSet;
  • if the end diameter and length are already defined, and the calculation of the transition angle is not between [1, 89] degrees.
