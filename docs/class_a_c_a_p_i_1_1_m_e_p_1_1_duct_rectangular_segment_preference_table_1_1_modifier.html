Result< void > ACAPI::MEP::DuctRectangularSegmentPreferenceTable::Modifier::SetSegmentationLengths ( double minimum,
double maximum )

Sets the segmentation lengths of the rigid segment. If the segmentation is turned off, the value is ignored when generating routings, and the rigid segment will be the same length as the routing segment.

Parameters
minimumThe minimum segmentation length of the rigid segment.
maximumThe maximum segmentation length of the rigid segment.
Returns
Returns error if any of the parameters are negative, or if the condition minimum * 2 <= maximum is false, otherwise nothing.
