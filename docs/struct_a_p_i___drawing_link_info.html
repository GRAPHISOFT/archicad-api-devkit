Detailed Description

Parameters of a drawing link.

Remarks
You can get information about a given drawing link using this structure with the ACAPI_Drawing_GetDrawingLink function. The linkPath location is used only for links having associated source file location; otherwise, this pointer is nullptr. The viewPath is filled out only for internal and external view type links, otherwise it is nullptr. If the link has been broken, then the linkTypeID parameter is @ref API_DrawingLink_InfoID. In this case the infoTypeID field shows the original type of the link.
