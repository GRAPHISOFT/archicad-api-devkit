Detailed Description

The possible parameter values of a non-array Library Part parameter.

Remarks
The possible parameter values for the given parameter (defined by its name or index) are returned in this structure by the ACAPI_LibraryPart_GetParamValues function. For more information, see the description of the VALUES command in the Parameter Script section of the GDL Reference Manual. Remember to dispose of the strValues or the realValues handles when no longer needed.
