Detailed Description

A SystemGroup represents a collection of PhysicalSystems.

The SystemGroup has its own name and can be used by a user in the system browser to categorize sets of PhysicalSystems. The PhysicalSystems contained in the SystemGroup does not need to share a common connection (e.g. domain).

// First, locate the default system group:
auto systemGroups { ACAPI::MEP::GetSystemGroupIDs () };
if (systemGroups.IsErr ())
return {};
for (const auto& id : systemGroups.Unwrap ()) {

if (systemGroup.IsErr ())
return {};
if (systemGroup->GetName () == DefaultSystemGroupName)
return systemGroup.Unwrap ();
}

return newlyCreatedSystemGroup.Unwrap ();
}
