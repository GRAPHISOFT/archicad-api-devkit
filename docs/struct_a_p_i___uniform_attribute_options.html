Detailed Description

General representation of a Model Display in 3D Document.

Remarks
You are able to get and change the default settings of 3D document via ACAPI_View_GetDocumentFrom3DDefaults and ACAPI_View_ChangeDocumentFrom3DDefaults functions. ACAPI_View_GetDocumentFrom3DSettings and ACAPI_View_ChangeDocumentFrom3DSettings function retrieves and modifies the settings of an existing 3D document database.
