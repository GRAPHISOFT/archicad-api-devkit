Detailed Description

The PreferenceTableContainerBase contains PreferenceTables, and you can Add/Get/Delete them.

Since
Archicad 27

The containers for the three MEP domains are already initialized and you can get them from DuctPreferenceTableContainer, PipePreferenceTableContainer and CableCarrierPreferenceContainer

Example code snippet from Test / New MEP

return ductSegmentContainer.UnwrapErr ().kind;
